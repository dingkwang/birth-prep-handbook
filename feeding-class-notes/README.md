# 婴儿喂养课讲义 · Baby Academy Feeding

根据 The Baby Academy 喂养课（讲师 Cris）中文整理的**单页静态讲义**。

## 打开

```bash
open index.html
# 或
python3 -m http.server 8080 --directory .
```

## 内容重点

- 全课：饥饿信号、肌肤接触、初乳、尿便、母乳/配方/混合、含乳、姿势、泵奶、拍嗝、维生素 D、Q&A
- **加细**：配方装备、冲调步骤、**足月前 2 个月热水灭菌**、保存时限、喂多少锚点（毫升）
